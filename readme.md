Proyecto WordPress - Cooperativa San Roque
Este proyecto forma parte de las prácticas de Ingeniería de Software 1. 
Consiste en un sitio institucional para la Cooperativa San Roque de Villa del Lago, desarrollado 
sobre un stack LAMP y gestionado mediante control de versiones.

Tecnologías Utilizadas
Servidor: Ubuntu Server (Virtual Machine).
Stack: Apache, MySQL, PHP (LAMP).
CMS: WordPress con Theme personalizado.
Versionado: Git & GitHub.

Instalación y Configuración
1. Clonado del repositorio
2. Configuración SSH:El acceso al servidor se realiza mediante túneles SSH configurados en VS Code.


Personalización del Theme
Se realizaron ajustes en la hoja de estilos (`style.css`) para optimizar la visualización de banners panorámicos, utilizando unidades relativas para asegurar un diseño responsivo.
